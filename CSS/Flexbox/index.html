Remember This :

- the align-items: stretch; and align-content: stretch; works only if the element does not have the size set in that direction (depending on the "main-flex-axis" current direction)

- float , clear and vertical-align does not have any effent on felxbox items!

- flex-wrap:wrap overwrites the flex-shrink property

Description and Demonstration :

- Flex boxes provide a very efficient way to distribute AVAILABLE space among items in a container, (flex-box is CSS level 3)

- The flex container is declared by display:flex; or display:inline-flex; properties

- Everything outside a flex container and inside a flex item are not affected by any of the flex properties

- The flex-box layout is "flex-line" oriented design, all elements are positioned relative to this line 

SYNTAX :    display: flex                   // block-level flex element          // specifies the display behavior of the element 
                     inline-flex            // inline flex element 
        
            flex-direction: row             // (default) left-to-righ              // sets the "main-flex-axis" direction 
                          : row-reverse     // right-to-left
                          : column          // top-to-bottom
                          : column-reverse  // bottom-to-top
                 
            flex-wrap: nowrap               // (default) flex-line doesn't wrap    // specifies if/how the "main-flex-axis" is wrappwed when the flex children have no more space along the "main-flex-axis"    
                     : wrap                 // wraps the flex-line when necessary
                     : wrap-reverse         // wraps the flex-line when necessary in inverse order
    
            flex-flow: flex-direction flex-wrap                                    //  shorthand property for the flex-direction and flex-wrap properties
                           
            order: nr                       // (0 default) element order           //  defines the order of the flex-container's child elements relative to each other (property is set on the child element)
                           
            flex-basis: cssLength                                                  //  sets the size of the element along the "main-flex-axis
                      : auto                // (default) equlas to the element's current size (width or height depending on the flex-line direction)
            flex-grow: nr                   // (default 0 not growing)             //  defines how much the item grows (along the "main-flex-axis) (relative to the "flex-basis" size) relative to the other elements in the same flex-container (relative to other elements' flex-grow property)
            flex-shrink: nr                 // (default 1) 0 means no shrink       //  defines how much the element shrinks (along the "main-flex-axis) (relative to the "flex-basis" size) relative to the other elements in the same flex-container (relative to other elements' flex-shrink property) (element shrinks only if the flex-container forces it get smaller than the flex-basis size) (element never shrinks smaller than "content + padding") (the flex-wrap property overrides this property)   
            flex: flex-grow flex-shrink flex-basis   // (default 0 1 auto)         //  shrothand property for the "flex-grow" "flex-shrink" and "flex-basis" properties

// perfectly centered element -----------------------------------------------------
    .myElement { display: flex; }
    .myElement > childElement { margin:auto; }                                     // child element perfectly centered
        
// flex-direction / flex-wrap / flex-flow / --------------------------------------- 
    .myElement {
        display: flex;
        fext-direction: row-reverse;                                               // element order (main-flex-axis) goes from right-to-left
        flex-wrap: wrap;                                                           // flex line is wrapped if the box is not large enough (along the "main-flex-axis") to the children elements
  // -> flex-flow: row-revers wrap;                                                   does the same as the two properties above (shorthand property)
    }
    
// order --------------------------------------------------------------------------  
// -! the "order" property must be set on ALL the elements otherwise the "counted elements" will be placed AFTER the uncounted ones (in this case "
3
 
2
 
1
")     
    .myElement div:first-child { order: 2; }                                       // element placed as 2nd in the element order 
    .myElement div:nth-child(2) { order: 1; }                                      // element placed as 1st in the element order 
                                                                                   // the "order" property works only on flex-container children elements
    
// flex-basis / flex-shrink / flex-grow / flex ------------------------------------ 
// -! "flex-grow" distributes the available space only! (space distribution is calucated above the original size)
// -! "flex-shrink" fires only if the flex-container forces its children elements to shrink but elements never get smaller than its content + padding
// !! if "flex-wrap" property is set elements never shrink because the flex-container will wrap the "main-flex-line" instead of shrinking elements 
    .myElement div:first-child {
        flex-basis: 50px;                                                          // element size along the "main-flex-axis" is set (in this case the width because the "main-flex-axis" goes from left-to-right)
        flex-grow: 0;                                                              // (default value) element is not growing (never gets bigger than 20px;)
        flex-shrink: 0;                                                            // element is not shrinking (never gets smallert than 20px;)
    }                                   
    .myElement div:nth-child(2) {       
        flex-basis: 60px;               
        flex-grow: 1;                                                              // element grow values is 1/6    // !! takes 1 part out of 6 from the available space
        flex-shrink: 2;                                                            // element shrinks 2 times faster relative to other elements 
    }
    .myElement div:nth-child(3) {
        flex-basis: 70px;                                                          // -! element sizes are not the same (for the demo sake)
        flex-grow: 3;                                                              // element grow values is 3/6    // !! takes 3 parts out of 6 from the available space
        flex-shrink: 1;                                                            // element shrinks 1 time faster relative to other elements (default value)
    }
    .myElement div:last-child {
        flex-basis: 50px;
        flex-grow: 2;                                                              // element grow values is 2/6    // !! takes 2 parts out of 6 from the available space
        flex-shrink: 4;                                                            // element shrniks 4 times faster than relative to other elements 
  // -> flex: 2 4 50px;                                                               does the same as the 3 properties above (shorthand property)   
    }
    // -! this is how the above "flex-grow" is calculated, (0+1+3+2=6) the available space is slit up to 6 pieces than the "flex-grow" property defines how much of this shared space is distributed to each individual element

            place-content: align-content justify-content                           // shorthand property for the align-content and justify-content properties   
                         : align-content/justify-content  // one value applies for both properties
                         
            aling-items: stretch            // (default) stretched along the "cross-flex-axis" (works only if the element has no height set)   //  defines the alignement of the flex-container's child elements along the "cross-flex-axis" (the align-content and align-self properties override this property)
                       : center             // centered along the "cross-flex-axis"
                       : flex-start         // placed at the start of the "cross-flex-axis"
                       : flex-end           // placed at the end of the "cross-flex-axis"
                       : baseline           // positioned at the baseline (text line) of the container 
                         
            align-self: auto                // (default) inherits the align-itmes property value from the parent element or "stretch" if there is no parent element    //  defines the alignement of the flex-container's child elements individually along the "cross-flex-axis" (property is set on the child element) (this property overrides the align-items property but it is overriden by the align-content property)   
                      : stretch             // stretched along the "cross-flex-axis" (works only if the element has no height set) 
                      : flex-start          // placed at the beginning of the "cross-flex-axis"
                      : flex-end            // placed at the end of the "cross-flex-axis" 
                      : baseline            // positioned at the baseline (text line) of the container 
    
// align-content / justify-content / place-content --------------------------------
    .myElement {
        display: flex;
        align-content: center;                                                     // the"main-flex-axis" is placed in the center of the "cross-flex-axis", when the it wraps the "main-flex-axes" are centered evenly   
        justify-content: space-evenly;                                             // space distributed evenly between the elements along the "main-flex-axis"  
    // ------------------------------------
        place-content: center space-evenly;                                        // does the same as the 2 properties above 
        
// align-items --------------------------------------------------------------------
        align-items: center;                                                       // all flex items are centered on the "main-flex-line"  
    }
    
// align-self ---------------------------------------------------------------------
    .myElement > chld {                                                            // -! property is set on the flex item 
        align-self: flex-start;                                                    // places this element at the start along the "cross-flex-axis"
