Property Value Description
colorName predefined names by CSS which expresses colors
#hexValue hexadecimal value rrggbb which defines a color code
rgb(nr, nr, nr) defines a color by the given (red, green, blue) values
rgba(nr, nr, nr, nr_A) defines a color by the given (red, green, blue) values and the opacity level
hsl(hue_nr, sat_%, lig_%) defines a color by the given (hue, saturation, light) parameters
hsla(hue_nr, sat_%, lig_%, nr_A) defines a color by the given (hue, saturation, light) parameters and the opacity level
opacity nr_A (0.0 - 1.0) defines the opacity level
filter alpha(opacity=nr_A) (0.0 - 1.0) defines the opacity level (IE8 and earlier)

- in CSS we have the following color values 

SYNTAX :    colorName                               // predefined names by CSS which expresses colors 
            #hexValue                               // hexadecimal value rrggbb which defines a color code
            
            rgb(nr, nr, nr)                         // defines a color by the given (red, green, blue) values 
            rgba(nr, nr, nr, nr_A)                  // defines a color by the given (red, green, blue) values and the opacity level 
                - nr : defines the color number 0 - 255
                - nr_A : opacity lever 0.0 - 1.0 (0 is fully opaque)
            
            hsl(hue_nr, sat_%, lig_%)               // defines a color by the given (hue, saturation, light) parameters
            hsla(hue_nr, sat_%, lig_%, nr_A)        // defines a color by the given (hue, saturation, light) parameters and the opacity level
                - hue_nr : hue degree 0 - 360
                - sat_% : saturation percent 0% - 100%
                - lig_% : light percent 0% - 100%
                - nr_A : opcity lever 0.0 - 1.0 (0 is fully opaque)
                
            opacity:nr_A;                           // defines the opacity level 
            filter:alpha(opacity=nr_A)              // defines the opacity lever (IE8 and earlier)
                - nr_A : opacity lever 0.0 - 1.0 (0 is fully opaque)
    
// same color different ways to display it -----------------

    myElement {
        color: cornflowerblue;                          // color name 
        color: #6495ed;                                 // hexadecimal value
        color: rgb(100, 149, 237)                       // rgb value
        color: hsl(218.5, 79.2%, 66.1%)                 // hsl value
    // control opacity (half opaque) ----------------------
        color: rgba(100, 149, 237, 0.5)                 // rgb value with opacity level
        color: hsla(218.5, 79.2%, 66.1%, 0.5)           // hsl value with opacity level
        opacity: 0.5;                                   // opacity level porperty set!
        filter: alpha(opacity=50);                      // -> IE8 and earlier
    }
