Description and Demonstration :

- Media Queries are an efficient technique for delivering tailored styles for different media types and screen sizes

SYNTAX:     @media not|only mediaType and (features) {}                      // if conditions are true CSS executes the code block

- Media Queries level 4 Syntax 

    @media all and (min-width:800px), not print {               // all devices at least 800px width excluding printers 
        color: green;
    }

Implement Mediaqueries (2 ways) 

// entire stylesheet dedicated to the targeted conditions (by using the "media" attribute) 
-- CODE (HTML) --
<head>
    <link rel="stylesheet" media="only print" href="styles-media.css">                            // sheet only implemented on printers
    <link rel="stylesheet" media="(max-width: 800px)" href="sytles-max-width-800px.css">          // sheet only implemented on devices less than 800 pixels
</head>
    

// mediaquery condition specified separately on the stylesheet
-- CODE (CSS) --
    @media screen and (max-width:500px) {
        // CSS code for the specified media device
    }
    // other CSS code

Operatros

the and operator

- the and operator combines multiple media features togehter, as well as combining media features with media types

    @media (min-width:700px) and (orientation:landscape) { }                // only applies if device screen is wider than 700px and landscape oriented 
    @media tv and (min-width:700px) and (orientation:landscape) { }         // this does the same but only applies ot TV screen

the only operator

- the only operator applies only on the explicit media type (partially deprecated)

- mostly used for preventing older browsers that do not support media queries with media features from applying the given styles

    @media only screen                                                      // applies only on screen

the not operator

- the not operator negates an entire media query (must specify an explicit media type) (partially deprecated) 

    @media not print { }                                                    // declarations won't apply on any printing devices

the ,(comma) operator

- the ,(comma) operator separates multiple media query conditions, each query is treated individually 

    @media (max-width: 500px), (min-width: 800px) { }                       // style only applies if screen width is smaller than 500px or larger than 800px

Media Query Features

width / min-width / max-width / height / min-height / max-height - sets the dimension of the display area such as browser window 

    @media screen and (min-width:500px) and (max-width:800px) and (min-height:350px) and (max-height:600px) { }

orientation (landscape or portrait) - orientation of the viewport 

    @media screen and (orientation:landscape) { }

color / min-color / max-color - can check if the device is color device or indicates the minimum and maximun number of bits per color component

Wikipedia color depth explained

    @media not (color) { }                                                  // for black and white screens only 
    @media (min-color:8) { }                                                // parameters will be used only on 8bits (256 colors) or higher screens --*/

color-index / min-color-index / max-color-index - indicates the number of colors the device can display 

    @media all and (color-index) { }                                        // targets color indexed displays (at least 256 colors) 
    @media (max-color-index:255) { }                                        // displays less than 256 colors

aspect-ratio / min-aspect-ratio / max-aspect-ratio - targeting or limiting aspect ratios (5:4 / 4:3 / 16:10 / 16:9)

Wikipedia aspect ratio explained

    @media screen and (aspect-ratio:16/9), screen and (aspect-ratio:16/10) { }  // targetes set aspect ratio
    @media screen and (min-aspect-ratio:1/1) { }                             // screens minimum square or wider 
    @media screen and (max-aspect-ratio:1/1) { }                             // screens at least square or narrower

resolution / min-resolution / max-resolution - the resolution of the output device using dpi or dpcm

    @media (min-dpi:300dpi) { }                                             // device at least 300dpi

simple example

(width=1000px) Background Color is green between 500px and 800px
(width=1000px) Background Color is green below 500px and above 800px
CODE : 
-- HTML --
    <div class="media-one"> (width=1000px) Background Color is green between 500px and 800px </div>
    <div class="media-two"> (width=1000px) Background Color is green below 500px and above 800px </div>
                
-- CSS --
    [class|=media]{
        background-color: orange;
        max-width: 1000px;
        height: 50px;
        margin: auto;
        text-align: center;
        border: 3px black solid;
    }
    @media screen and (max-width:800px) and (min-width:500px){
        .media-one{
            background-color: green; 
        }
    }
    @media screen and (max-width:500px), screen and (min-width:800px){
        .media-two{
            background-color: green;
        }
    }
