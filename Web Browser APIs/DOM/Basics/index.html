Remember This :

- the space between DOM elements are considered as text nodes! (ex: <p>   <span></span>   </p>, this paragraph has 3 child nodes!)

- comment nodes don't have embeded text nodes like elements!

Description and Demonstration :

- the HTML DOM (Document Object Model) is a defined standard by W3C for accessing documents and elements in web page

- the HTML DOM is a standard object model and programing interface for HTML, it gives all the power to JavaScript (and other programs) to do basically modify everything in the page

- there are 3 DOM standard, / Core DOM / HTML DOM / XML DOM /

- HTML DOM elements are defined as objects in JavaScript, each object has properties and methods

DOM Elements VS DOM Nodes

- The DOM tree can be seen as build up of HTML Elements but it also can be seen that is built up of DOM Nodes

- According to "W3C"

  • element node is an element
  • text node is the text inside an HTML element
  • attribute node is an attribute (Deprecated!)
  • comment node is a comment

- the Node navigation system gives the developer a different way (methods and properties) to manage the DOM tree

- on the DOM three elements have spaces between them (like the lines on the image) these are also considered as nodes (text nodes)! (ex: <p>   <span></span>   </p> has 3 child nodes!)

- HTML Elements or HTML collections and DOM Nodes or Node List are basically the same things just the way we access to the elements change

- comment Nodes have no embeded textNodes!

Encapsulation & Shadow DOM

- Encapsulation means keeping a markup (structure / style / behaviour) separate and hidden from the main script

- the Shadow DOM is "encapsulated" form the main markup, a good example is the video player, it has its own markup but the developer cannot see this DOM structure




















HTML DOM CSS reference

- creating nodes are faster with direct methods (ex: createElement()) than parsing a string to HTML code (ex: innerHTML)

HTML Collection VS Node List

The length / item() / namedItem() / entries() / keys() / values() / forEach() property and methods

- an HTMLCollection is an Object containing multiple HTML elements, collected by an HTML collector like children

- a Node List is an Object containing multiple Nodes, collected by a Node collector like childNodes

SYNTAX :    HTMLCollection|NodeList.length                                             // returns the number of elements in the collection
            
            HTMLCollection|NodeList.item(nr:index)                                     // returns the indexed element|node from the collection
            HTMLCollection.namedItem(name|id:str)                                      // references an element from the collection by its id or name attribute value and returns it    
            
            NodeList[nr:index]                                                         // returns the indexed node from the collection
            HTMLCollection[nr:index|id|name:str]                                       // references an element from the collection by its id or name attribute value and returns it, also can be referenced by index    
            
            NodeList.entries()                                                         // returns an iterable object with the NodeList element's key/value (index/node) pairs
            NodeList.keys()                                                            // returns an iterable object with the NodeList element's keys (indexes) 
            NodeList.values()                                                          // returns an iterable object with the NodeList element's values (nodes) 
            
            NodeList.forEach(callbackFunction(value, key, NodeList), thisValue)        // executes a function once per node
                - callbackFunction(value, key, NodeList) : function executed at each iteration
                    - value : the current element's value 
                    - key : the current element's key
                    - NodeList : the NodeList itself 
                - thisValue : referes to the this keyword in the callback function
    
    <p> elso elem </p>
    <p id="2nd"> masodik elem </p>
    <p name='3rd'> harmadik elem </p>
    <p> negyedig elem </p>
<script>
    var myHTMLCollection = document.getElementById('demoList').getElementsByTagName("li");  // HTMLCollection 
    var myNodeList = document.getElementById('demoList').querySelectorAll("li");            // NodeList
    
// length -----------------------------------------------------------------------------
    myHTMLCollection.length;                                                           // -> 4      // collection has 4 elements
    myNodeList.length;                                                                 // -> 4      // list has 4 nodes
    
// item() / namedItem() ---------------------------------------------------------------
    myHTMLCollection.item(1);                                                          // -> <p id="2nd"> masodik elem </p>     // element indexed  
    myNodeList.item(2);                                                                // -> <p name='3rd'> harmadik elem </p>  // element indexed  
    
    myHTMLCollection.namedItem('2nd');                                                 // -> <p id="2nd"> masodik elem </p>     // element referenced by id attribute (HTMLCollection only!)  
    myHTMLCollection.namedItem('3rd')                                                  // -> <p name='3rd'> harmadik elem </p>  // element referenced by name attribute (HTMLCollection only!)   
     
// [ ] --------------------------------------------------------------------------------
    myNodeList[2];                                                                     // -> <p name='3rd'> harmadik elem </p>  // node list items can be indexed only   
    
    myHTMLCollection[1];                                                               // -> <p id="2nd"> masodik elem </p>     // HTML collection items can be indexed
    myHTMLCollection['2nd'];                                                           // -> <p id="2nd"> masodik elem </p>     // HTML collection items can be referenced by id attribute 
    myHTMLCollection['3rd'];                                                           // -> <p name='3rd'> harmadik elem </p>  // HTML collection items can be referenced by name attribute    
    
// entries() / keys() / values() ------------------------------------------------------
    var getEntries = myNodeList.entries();                                             // returns an iterable object containing the lists key value (index node) pairs 
    getEntries.next();                                                                 // -> {value:[0, <p> elso elem </p>], done:false}   
        
    var getKeys = myNodeList.keys();                                                   // returns an iterable object containing the lists keys    
    getKeys.next();                                                                    // -> {value:0, done:false}    
        
    var getValues = myNodeList.values();                                               // returns an iterable object containing the lists values    
    getValues.next();                                                                  // -> {value:<p> elso elem </p>, done:false}   
    
// forEach() -------------------------------------------------------------------------- 
    myNodeList.forEach(function(value, key, nodeList){                                 // runs a pass for each node in the list 
        value;                                                                         // -> <p> elso elem </p> | ...       // node 
        key;                                                                           // -> 0                  | ...       // index
        nodeList;                                                                      // -> NodeList(4)        | ...       // the node list itself   
    });
</script>

The parentNode / childElementCount / children / childNodes / firstElementChild / firstChild / lastElementChild / lastChild / nextElementSibling / nextSibling / previousElementSibling / previousSibling / elements properties 

SYNTAX :    element.parentNode	                                                       // returns the element's parent Node
            element.childElementCount	                                               // returns the number of elements of the parent element as a number

            element.children[index]                                                    // returns an HTML collection of the element's children (optionally can access them)
            element.childNodes[index]                                                  // returns a NodeList of the element's children (optionally can access them)
	
            element.firstElementChild	                                               // returns the first child element 	
            element.firstChild	                                                       // returns the first child Node 
            element.lastElementChild                                                   // returns the last child element
            element.lastChild	                                                       // returns the last child Node

            element.nextElementSibling	                                               // returns the next element	
            element.nextSibling	                                                       // returns the next Node	
            element.previousElementSibling                                             // returns the previous element
            element.previousSibling	                                               // returns the previous Node
            
            formElement.elements[index]                                                // returns an HTML Collection from a form element's children

<script>
    var parent = document.getElementById("parent");

    document.getElementById('child').parentNode;                                       // -> "<div id="parent">"
    parent.childElementCount;                                                          // -> 3
    
// children / childNodes --------------------------------------------------------------
    parent.children;                                                                   // -> [p, p#child, p, child: p#child]                           // HTMLCollection
    parent.children[2];                                                                // -> <p> child paragraph-3 </p>
    parent.childNodes;                                                                 // -> [text, p, text, p#child, text, comment, text, p, text]    // NodeList
    parent.childNodes[5];                                                              // -> <!-- comment -->

    parent.firstElementChild                                                           // -> <p> child paragraph-1 </p>
    parent.firstChild                                                                  // -> #text                                                     // the first Node is returned!
    parent.lastElementChild                                                            // -> <p> child paragraph-3 </p>
    parent.lastChild                                                                   // -> #text                                                     // the last Node is returned!
    
// nextElementSibling / nextSibling / previousElementSibling / previousSibling --------
    document.getElementById("child").nextElementSibling                                // -> <p> child paragraph-3 </p>
    document.getElementById("child").nextSibling                                       // -> #text                                                     // the next Node is returned!
    document.getElementById("child").previousElementSibling                            // -> <p> child paragraph-1 </p>
    document.getElementById("child").previousSibling                                   // -> #text                                                     // the previous Node is returned!
    
// elements ---------------------------------------------------------------------------
    document.forms['demo'].elements.lenth;                                             // -> 3

Size / Position / Scorolling offsetHeight / offsetWidth / clientHeight / clientWidth / scrollWidth / scrollHeight / getBouncingClientRect() / scrollTop / scrollLeft

SYNTAX:         element.offsetWidth             // returns the element's width (element + padding + border)
                element.offsetHeight            // returns the element's height (element + padding + border)
                
                element.clientWidth             // returns the element's width (element + padding)
                element.clientHeight            // returns the element's height (element + padding)
                
                element.scrollWidth             // returns the element's width (element + padding [overflowing content aslo returned])
                element.scrollHeight            // returns the element's height (element + padding [overflowing content aslo returned])
        
                element.getBouncingClientRect() // returns an object containing the element's size and its position relative to the viewport (returned object properties are top / bottom / left / right / x / y / width / height)    
                
                element.scrollTop               // returns or sets an element's vertical scrolling position
                element.scrollLeft              // returns or sets an element's horizontal scrolling position
