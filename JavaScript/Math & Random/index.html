The floor() / ceil() / round() methods 

SYNTAX :    Math.floor(val)                                                            // converts the passed value to a number and rounds its value to its lowest integer and returns the result    
            Math.ceil(val)                                                             // converts the passed value to a number and rounds its value to its highest integer and returns the result
            Math.round(val)                                                            // converts the passed value to a number and rounds its value to its closest integer and returns the result (.49 rounded down | .50 rounded up)   
    
// floor() / ceil() -------------------------------------------------------------------
    Math.floor(2.54);                                                                  // -> 2
    Math.ceil(1.14);                                                                   // -> 2
    
// round() ----------------------------------------------------------------------------
    Math.round(3.49);                                                                  // -> 3
    Math.round(3.50):                                                                  // -> 4

The min() / max() methods 

SYNTAX :    Math.min(val, ...)                                                         // converts the passed arguments to numbers and returns the one with the lowest value (returns NaN if any converted argument has a NaN value)    
            Math.max(val, ...)                                                         // converts the passed arguments to numbers and returns the one with the largest value (returns NaN if any converted argument has a NaN value)    
    
// min() ------------------------------------------------------------------------------
    Math.min(0, 54, 7);                                                                // -> 0
    Math.min(NaN, 45, -75);                                                            // -> NaN        
    Math.min(-7854, -Infinity, 447);                                                   // -> -Infinity
    
// max() ------------------------------------------------------------------------------
    Math.max(-45, -47, -31, -12);                                                      // -> -12
    Math.max(false, 32, [56]);                                                         // -> 56         // all arguments can be converted to a number 
    Math.max(1, 2, 'three');                                                           // -> NaN        // cannot convert all arguments to a number, NaN returned

The pow() / sqrt() / cbrt() methods 

SYNTAX :    Math.pow(base:any, exponent:any)                                           // converts the passed arguments to numbers and takes the base to the power of exponent and returns the result  
            
            Math.sqrt(val)                                                             // converts tha passed value to a number and returns the square root value of it (number what is takent to the power of 2 results this number)  
            Math.cbrt(val)                                                             // converts tha passed value to a number and returns the cube root value of it (number what is takent to the power of 3 will results this number)    
    
// pow() ------------------------------------------------------------------------------
    Math.pow(2, 5);                                                                    // -> 32 
    Math.pow([2], '5');                                                                // -> 32                     // arguments are converted to numbers 
    
// sqrt() -----------------------------------------------------------------------------
    Math.sqrt(49);                                                                     // -> 7 
    Math.sqrt('1024');                                                                 // -> 32                     // value converted to a number 
    
// cbrt() -----------------------------------------------------------------------------
    Math.cbrt(343);                                                                    // -> 7 
    Math.cbrt('32768');                                                                // -> 32                     // value converted to a number

The abs() / trunc() / sign() / fround() / clz32() methods 

SYNTAX :    Math.abs(val)                                                              // converts the passed value to a number and returns the absolute value of it (no negative values allowed, they are converted to positive values)    
            
            Math.trunc(val)                                                            // converts the passed value to a number and returns the integer part of it (chops off decimals)    
            Math.sign(val)                                                             // converts the passed value to a number, returns 1 for a positive value | returns -1 for a negative value | returns 0 for 0    
            
            Math.fround(val)                                                           // converts the passed value to a number and returns the closest 32bit single precision floating point representation of it    
            Math.clz32(val)                                                            // converts the passed value to a number and returns the number of leading zero bits in the 32bit binary representation of this number     
    
// abs() ------------------------------------------------------------------------------
    Math.abs('-1');                                                                    // -> 1
    Math.abs(-2);                                                                      // -> 2
    Math.abs(null);                                                                    // -> 0
    Math.abs('');                                                                      // -> 0
    Math.abs([]);                                                                      // -> 0
    Math.abs([2]);                                                                     // -> 2
    Math.abs([1,2]);                                                                   // -> NaN
    Math.abs({});                                                                      // -> NaN
    Math.abs('string');                                                                // -> NaN
    Math.abs();                                                                        // -> NaN
    
// trunc() / sing() -------------------------------------------------------------------
    Math.trunc('32.32');                                                               // -> 32                     // returns the integer part only  
    Math.trunc([142]);                                                                 // -> 142
    Math.trunc(-415.22);                                                               // -> -415
    
    Math.sign([84552.21]);                                                             // -> 1                      // positive value 
    Math.sign(-441.2);                                                                 // -> -1                     // negative value 
    Math.sign(0);                                                                      // -> 0
    
// fround() ---------------------------------------------------------------------------
    Math.fround(12.12);                                                                // -> 12.119999885559082     // closest 32bit single precision floating point representation    
    Math.fround(1.2);                                                                  // -> 1.2000000476837158
    
// clz32() ----------------------------------------------------------------------------
    Math.clz32(45);                                                                    // -> 26                     // number of leading 0bits in 32bit binary representation    
        // 00000000000000000000000000101101                                                // 32bit binary representation of 45 
    Math.clz32(5548796);                                                               // -> 9
        // 00000000010101001010101011111100                                                // 32bit binary representation of 5548796
    Math.clz32(-125);                                                                  // -> 0                      // negative number has the leftmost bit (sign bit) 1 
        // 10000000000000000000000001111101                                                // 32bit binary representation of -125
