Method Description
console.assert(exp|message) prints the error in the console if the passed expression is evaluated "false" (otherwise does nothing) 0.0.101

console.debug(obj|str) alias for console.log() 8.0.0 alias console.log() 8.1.0
console.dir(obj) displays the passed object in "Object View" in the console window 0.1.101
console.dirxml(obj) displays the passed object in "Interactive Markup View" if possible, otherwise in object view (does the same as log [TESTED 2019-02-09]) 8.0.0 alias console.log() 9.2.0
console.error(obj|message) outputs the passed message as an error in stderr 0.1.100
console.group("label") creates a message group in the console 8.5.0
console.groupCollapsed("label") creates a collapsed message group in the console (same as group() but collapsed by default) 8.5.0
console.groupEnd("label") ends the message group in the console created by the group() or groupCollapsed() methods 8.5.0
console.info(obj|message) prints the passed informational message in the console (same as console.log()) 0.1.100
console.log(obj|message) prints the passed message in the stdout 0.1.100
console.time("label") start the console execution timer counter 10.0.0
console.timeEnd("label") ends the console execution timer counter and outputs to the stdout the result 0.1.100
console.timeLog("label") outputs the current execution timer to the stdout started by the time() method 10.7.0
console.table(obj|arr, ["col", ...]) prints a table in the console which visually represents the passed object (can restrict which columns to print) 10.0.0
console.trace("label") outputs the stack trace to the stderr 0.1.104
console.warn(obj|message) alias for console.error() 0.1.100
