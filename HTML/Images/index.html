Image Map (Client-Side) <map> / <area>

- we can specify areas on an image as clickable areas acting as hyperlinks with the help of map and area tags 

SYNTAX :    <img src="imageSource" alt="text" usemap="myMap"> 
            <map name="myMap">
                <area shape="rect" coords="nr,nr,nr,nr">
                <area shape="circle" coords="nr,nr,nr">
                <area shape="poly" coords="nr,nr,nr,nr,nr,nr,nr,nr,nr,nr,nr,nr">
            </map>
    
    <img src="myImage.jpg" alt="image-map" usemap="#angryBirds">          // the "usemap" attribute targets "map" tag which describes the clickable areas
    <map name="angryBirds">
        <area shape="rect" coords="2,2,100,100" href="demoSite.html" alt="demoTarget">
            // rectangle clickable area (coords="2,2,100,100" [2,2 = top left corner coordinates] [100,100 = bottom rigth corner coordinates])
        <area shape="circle" coords="75,75,20" href="demoSite.html" alt="demoTarget">
            // circle clickable area (cords="75,75,30" [75,75 = coordinates of the circle center] [30 = defines the circle radius size])    
        <area shape="poly" coords="2,2,40,40,45,100,20,75" href="Images/angry_birds.png" alt="angryBirds" download="CusomImageName">
            // polygon clickable area (cords="2,2,40,40,45,100,20,75" [every number pair "2,2" "40,40" "45,100" "20,75"] defines polygon edges) 
            // "angry_birds.png" will be downloaded and renamed to "angyrBirds"
    </map>

Define image in HTML5 (additinal way not replacement) <picture> / <source>

- in HTML5 we can use the <picture> tag along with the <source> tags to display images (same system as <video> and <audio> tags)

- the responsive webdesign benefits this system because we can declare multiple images in the same tag and define different screen sizes as "mediaQueries" for each image, this way different images will be displayed on different screens sizes 

SYNTAX :    <picture>
                <source srcset="img" media="mediaQuery">   
                <source srcset="img" media="mediaQuery">  
                <img src="imageSource" alt="text">                // the only targetable element by CSS in order to style the images [2019-03-31]         
            </picture>    
    
    <picture>
        <source srcset="image_1.jpg" media="(max-width:500px)">   // if screen size is less than 500px this source will be displayed (display precedence order is form top to bottom)
        <source srcset="image_2.jpg" media="(min-width:501px)">   // if screen size is higher than 501px this source will be displayed
        <img src="backupImage.jpg" alt="backup Image" id="target">// backup image (used for backward compatibility)   // !! only the "img" tag can be targeted by CSS which styles will be applied to all picture sources   
    </picture>
Example :

display different pictures on different window sizes (resize to see)

CODE 
        <picture>
            <source srcset="Images/pink_flowers.jpg" media="(min-width:500px)" type="image/jpg">
            <source srcset="Images/white_flower.jpg" media="(min-width:450px)" type="image/jpg">
            <img src="Images/orange_flowers.jpg" alt="orange flowers" style="width:auto;">
        </picture>
