Tag / Property Attribute Description
<b> defines a bold text (use as last resort)

<del> cite="URL" (URL that explains why the text was deleted)
datetime="YYYY-MM-DDTHH:MM:SSZ" (time when text was deleted) 		defines a deleted text
<em> defines an emphasized (italic) text
<hr> defines a thematic break (draws a horizontal (separation) line on the page)
<i> defines an italic text (use as last resort)
<ins> cite="URL" (URL that explains why the text was inserted)
datetime="YYYY-MM-DDTHH:MM:SSZ" (time when text was inserted/changed) 		defines an inserted text
<kbr> defines a keyboard input
<mark> defines a marked/highlighted text
<s> specifies a text what is no longer correct (text line through)
<samp> defines a sample output from a computer program
<small> defines a smaller text and side comments
<span> groups inline elements (has no styling effects used to isolate and target text part)
<strong> defines an important (bold) text
<sub> defines a subscript text (text appears half a character below the normal line)
<sup> defines a superscript text (text appears half a character above the normal line)
<template> holds its content hidden form the client in a separate document fragment (its content is only accessible through the content property)
templateElement.content returns the content of the template element as a document fragment
<u> specifies a different text than the normal text (underlined)
<var> defines a variable (italic text)
<wbr> (word break opportunity) specifies a position in a text where a word-break would be ok 1 3 11.7 12 4

the <template> element

- the <template> element's content is not rendered when the page loads but it is parsed

- the <template> element's content is only accessible through the content property of the template object by using JS, (not accessible by CSS or direct targeting!) 

SYNTAX :    <template>                                    //  holds its content hidden form the client in a separate document fragment (its content is only accessible through the content property)   
            templateElement.content                       // returns the content of the template element as a document fragment 
    
    <template id="myTemplate">
        <p id="first"> first element </p>                 // all neseted elements are hidden from the client
        <p> second element </p>
    </template>
    
<script>
    var myTemplate = document.getElementById("myTemplate"); 
    myTemplate.style.visibility = "visible";              // this doesn't do anything (not hidden with CSS)
    document.getElementById("first");                     // -> null                                // child elements cannot be targeted directly! 
    
    myTemplate.content.nodeType;                          // -> 11                                  // 11 = document fragment
    myTemplate.content.children[0]                        // -> <p id="first"> first element </p>   // first child of the document fragment
    myTemplate.content.children[1]                        // -> <p> second element </p>             // second child of the document fragment
</script>
