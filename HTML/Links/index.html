Attribute / Pseudo Class Values Description
<a></a> tag attributes
download file_name (renames the downloaded file) (new in HTML 5) forces the targeted file to download
href URL
#id (target local page [bookmark])
javascript:JS code (executes a javaScript code) 		specifies the URL of the targeted page
hreflang language_code specifies the language of the targeted page
media media_query (new in HTML 5) specifies what media/device the linked document is optimized for
name name (NOT supported in HTML 5 use id instead) specifies the anchor's name
ping URL URL specifies a list of URL's to which POST requests with the body PING will be sent by the browser (in the background) when the hyperlink is clicked (used for tracking)
rel alternate (alternated representation translated or mirror)
author (author document)
bookmark (bookmark document)
external (external document (not part of the current site)
help (help document)
license (copyright document)
next (next document in the series)
nofollow (unendorsed document (like paid link used by Google)
prev (previous document in a series)
search ((search tool) 		specifies the relation between the current document and the tageted document (informative usage, mostly used by search engines)
target _blank (new window or tab)
_self (same frame the link is on)
_parent (parent frame)
_top (in the full body of the window)
frame_name (in the targeted frame) 		specifies where to open the linked document
type media_type (MIME) specifies the media type (purely advisory with no functionality)

Remember This :

Description and Demonstration:

- the <a></a> (anchor) tag creates a hyperlink which allowes the user to navigate from a page to another

- hyperlinks can be used as bookmarks or can target documents to open within the browser window or download them (like .pdf files)

- ancors can also execute JavaScript code 

SYNTAX :    <a attribute="value"></a>
    
    <a href="https://developer.mozilla.org/en-US/" target="_blank" title="Mozilla Developer Network"> MDN </a>      // opens MDN site
    
    <a href="javascript:void(document.body.style.backgroundColor='red')"> RED </a>                                  // executes a javaScript code
                                                                                                                    // "void" operator used to make sure the returned value is "undefined"
